1. Linear Search
Implement a function that performs linear search on an array of integers.

Input: an array and a target value.

Output: index of the target if found, otherwise -1.

Requirement: Test with at least 3 different arrays (small, medium, large).

2. Binary Search
Implement binary search on a sorted array of integers.

Input: a sorted array and a target value.

Output: index of the target if found, otherwise -1.

Requirement: Compare the number of comparisons made with linear search for the same dataset.

3. Big-O Notation Examples
Write code snippets that demonstrate:

O(1): Accessing an element in an array by index.

O(n): Linear search through an array.

O(n²): Nested loops (e.g., printing all pairs in an array).

Requirement: Explain in comments why each example fits its complexity class.

4. Queue
Implement a queue using an array or linked list.

Support operations: enqueue, dequeue, peek, isEmpty.

Requirement: Demonstrate with a scenario (e.g., ticket counter simulation).

5. Stack
Implement a stack using an array or linked list.

Support operations: push, pop, peek, isEmpty.

Requirement: Use the stack to reverse a string entered by the user.

6. Recursion
Write a recursive function for:

Factorial of a number.

Fibonacci sequence.

Requirement: Compare recursive vs iterative solutions in terms of readability and performance.

7. Single Linked List
Implement a singly linked list with operations:

Insert at head, insert at tail.

Delete a node by value.

Traverse and print all elements.

Requirement: Demonstrate by building a list of student names and performing insert/delete operations.

8. Double Linked List
Implement a doubly linked list with operations:

Insert at head, insert at tail.

Delete a node by value.

Traverse forward and backward.

Requirement: Demonstrate by storing integers and showing traversal in both directions.
