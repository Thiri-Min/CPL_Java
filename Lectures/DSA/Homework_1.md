## Homework Task: Linear Search

Objective:  
Implement a function that performs a linear search on an array of integers.

Requirements:

Input:

Accept an array of 10 integers entered by the user.

Accept a target integer value to search for.

Process:

Traverse the array sequentially.

Compare each element with the target value.

Output:

Return the index of the target if found.

Return -1 if the target is not present in the array.

Additional Notes:

Indices should start at 0 (zero-based indexing).

Print the array before performing the search to confirm user input.

Test with at least two different target values: one that exists in the array and one that does not.
