# Linear Data Structures

## Array / Dynamic Array
- **Array:** Fixed-size, contiguous memory storage.
- **Dynamic Array:** Resizable version (e.g., Python list, Java ArrayList).
- **Pros:** Fast access (O(1)), cache-friendly.
- **Cons:** Insertion/deletion costly (O(n)).

---

## Linked List
- **Types:** Singly, Doubly, Circular.
- **Pros:** Efficient insertion/deletion (O(1) at head/tail).
- **Cons:** Slow access (O(n)), extra memory for pointers.

---

## Stack
- **Definition:** LIFO (Last In, First Out).
- **Operations:** Push, Pop, Peek.
- **Use Cases:** Function calls, undo operations.

---

## Queue / Deque
- **Queue:** FIFO (First In, First Out).
- **Deque:** Double-ended queue (insert/remove at both ends).
- **Variants:** Circular Queue, Priority Queue.
- **Use Cases:** Scheduling, buffering, breadth-first search.

---

## Use Cases
- **Array:** Index-based access, lookup tables.
- **Linked List:** Dynamic memory allocation, real-time insertion.
- **Stack:** Expression evaluation, backtracking.
- **Queue/Deque:** Task scheduling, BFS traversal.
