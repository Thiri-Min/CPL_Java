Problem Description  
You are required to create a Maven-based Java console application. The project should demonstrate proper naming conventions, clean structure, layered architecture, and a simple domain model. The goal is to simulate a small real-world application (e.g., a student management system or library system) while applying best practices in project organization.

---

Part 1: Maven Project Setup  
Questions to Answer  
1. a) [20%]  
• Create a new Maven project using the standard directory structure (`src/main/java`, `src/test/java`).  
• Configure `pom.xml` with at least one external dependency (e.g., JUnit for testing).  
• Provide screenshots of successful project creation and build execution.  

---

Part 2: Console Program Implementation  
Questions to Answer  
2. b) [20%]  
• Implement a console-based program (e.g., Student Management System).  
• Input: user commands (add student, list students, search student).  
• Output: printed results in the console.  
• Add at least 10 test cases demonstrating functionality.  

---

Part 3: Clean Naming & Structure  
Questions to Answer  
3. c) [20%]  
• Apply proper naming conventions (PascalCase for classes, camelCase for methods/variables).  
• Ensure methods are short and readable (max 30 lines).  
• Provide examples of refactoring long methods into smaller helper methods.  

---

Part 4: Layered Architecture  
Questions to Answer  
4. d) [20%]  
• Organize the project into layers:  
  - **Domain Layer** (entities/models)  
  - **Service Layer** (business logic)  
  - **Repository Layer** (data access, in-memory or file-based)  
  - **Presentation Layer** (console interaction)  
• Demonstrate how layers interact with each other.  
• Provide screenshots of execution results showing layered calls.  

---

Part 5: Domain Model Design  
Questions to Answer  
5. e) [20%]  
• Design a simple domain model (e.g., `Student` with fields: id, name, age, grade).  
• Implement encapsulation with private fields and public getters/setters.  
• Show how the domain model is used in the console program.  
• Provide UML diagram or class diagram of the domain model.
