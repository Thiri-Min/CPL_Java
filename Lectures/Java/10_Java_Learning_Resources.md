# Java Learning Resources

## 1. Practice Websites 
- **W3Schools Java Tutorial** – Beginner-friendly tutorials with examples and online editor.  
  [https://www.w3schools.com/java](https://www.w3schools.com/java)

- **GeeksforGeeks Java Programming** – Explanations, coding problems, and practice challenges.  
  [https://www.geeksforgeeks.org/java](https://www.geeksforgeeks.org/java)

- **HackerRank (Java Domain)** – Practice coding problems, algorithms, and data structures.  
  [https://www.hackerrank.com/domains/tutorials/10-days-of-java](https://www.hackerrank.com/domains/tutorials/10-days-of-java)

- **LeetCode** – Advanced coding challenges and interview preparation.  
  [https://leetcode.com](https://leetcode.com)

## 4. Online Code Editors (Front-End & Back-End)
- **Replit** – Full-stack online IDE supporting Java, Python, Node.js, and front-end frameworks.  
  [https://replit.com](https://replit.com)

- **Glitch** – Collaborative editor for front-end and back-end web apps (Node.js).  
  [https://glitch.com](https://glitch.com)

- **CodeSandbox** – Specialized for front-end frameworks (React, Vue, Angular) with backend integration.  
  [https://codesandbox.io](https://codesandbox.io)

- **JSFiddle** – Lightweight editor for front-end (HTML, CSS, JavaScript).  
  [https://jsfiddle.net](https://jsfiddle.net)

- **Paiza.IO** – Online compiler supporting multiple languages including Java, Python, C++, and more.  
  [https://paiza.io](https://paiza.io)

- **Ideone** – Online compiler and debugging tool for 60+ languages including Java.  
  [https://ideone.com](https://ideone.com)

- **GitHub Codespaces** – Cloud-based VS Code environment for full-stack development.  
  [https://github.com/features/codespaces](https://github.com/features/codespaces)
