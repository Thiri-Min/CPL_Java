### Part B: Coding Tasks (OOP-focused)

Apply encapsulation, inheritance, and polymorphism.

Tasks:

Implement a Calculator class demonstrating variables, data types, and control flow.

Create a Student class with methods to calculate GPA.

Build a base class Vehicle with subclasses Car and Bike to demonstrate inheritance and polymorphism.
