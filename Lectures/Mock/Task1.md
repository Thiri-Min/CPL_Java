## Task:  

### Create a webpage using HTML, CSS, and Bootstrap with the following requirements

1. Navbar

- Logo aligned to the left corner.

- Navigation links: Home, About, Contact.

- A Login/Signup button aligned to the right corner.

2. Hero Section

- A large banner area with a headline and a subheadline.

- A call-to-action button.

3. Footer

- Footer should always be at the bottom of the page.

- Include copyright text and social media icons.

- Scroll-to-Top Arrow

- A floating arrow button that appears when the user scrolls down.

- Clicking the arrow should smoothly scroll back to the top.

4. Login/Sign up
- login/singup page using switch function and after signup or login, show alert.
- "You registered successfully!" for sign up 
- "Welcome to home page!" for login.
